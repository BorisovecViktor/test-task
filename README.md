# Phone store

[DEMO PAGE](https://borisovecviktor.github.io/react_phone-catalog/)

## Description
Product store implemented following Figma layout [this design](https://www.figma.com/file/uEetgWenSRxk9jgiym6Yzp/Phone-catalog-redesign?node-id=1%3A2).

### Technologies
- React/Redux
- HTML5/CCS3, Sass, BEM, Responsive Layout
- Typescript

### Additional Libraries
- Redux Trunk
- React Router
- React Transition Group

### JSON files
Used [products](https://borisovecviktor.github.io/react_phone-catalog/api/products.json) and [product details](https://borisovecviktor.github.io/react_phone-catalog/api/products/motorola-xoom.json) t–æ fetch data
